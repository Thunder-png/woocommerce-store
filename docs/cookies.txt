ÇEREZ POLİTİKASI
1. Çerez Nedir?

Çerezler (cookies), bir web sitesini ziyaret ettiğinizde tarayıcınız aracılığıyla cihazınıza kaydedilen küçük veri dosyalarıdır.

Bu dosyalar, web sitesinin daha verimli çalışmasını ve kullanıcı deneyiminin geliştirilmesini sağlar.

2. Kullanılan Çerez Türleri
Zorunlu Çerezler

Web sitesinin temel fonksiyonlarının çalışması için gereklidir.

Performans Çerezleri

Site kullanımını analiz etmek ve site performansını artırmak amacıyla kullanılır.

İşlevsel Çerezler

Kullanıcı tercihlerini hatırlamak için kullanılır.

3. Çerezlerin Kullanım Amaçları

Çerezler aşağıdaki amaçlarla kullanılabilir:

site güvenliğini sağlamak

kullanıcı oturumlarını yönetmek

site performansını analiz etmek

kullanıcı deneyimini geliştirmek

4. Çerez Yönetimi

Kullanıcılar tarayıcı ayarları üzerinden çerezleri kabul edebilir, reddedebilir veya silebilir.

Ancak bazı çerezlerin devre dışı bırakılması sitenin bazı fonksiyonlarının çalışmamasına neden olabilir.

5. Değişiklikler

Çerez politikası gerektiğinde güncellenebilir. Güncel metin web sitesi üzerinden yayınlanacaktır.
